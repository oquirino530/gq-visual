# Portfólio — Fotografia Esportiva

Arquivos criados para um site estático (HTML + CSS) de portfólio profissional e jovial.

Arquivos:

- [index.html](index.html) — página principal com seções Home, Portfólio, Sobre e Contato
- [styles.css](styles.css) — estilos principais e responsivos

Como personalizar rapidamente:

1. Substitua as imagens: os cards usam imagens do Unsplash como placeholder. Troque os `src` por caminhos locais `assets/photo1.jpg` ou URLs próprias.
2. Atualize seu nome, email e links em `index.html`.
3. Para publicar, basta enviar os arquivos para um servidor estático ou usar GitHub Pages.

Abrir localmente:

```
# No Windows Explorer ou terminal, abra o arquivo index.html no navegador
start index.html
```
